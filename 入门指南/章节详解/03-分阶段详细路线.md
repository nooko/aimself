### 核心概念速览

#### 神经网络本质

```
输入数据 → [权重矩阵1] → 激活函数 → [权重矩阵2] → ... → 输出预测
                                    ↑
                              训练 = 调整这些权重让预测更准
```

对游戏工程师的类比：
- **神经网络** ≈ 一个超大的参数化函数
- **训练** ≈ 用大量数据自动调参（不是手调配置表）
- **Loss** ≈ 预测值和真实值的距离（越小越好）
- **梯度下降** ≈ 数值优化（像物理引擎里求稳定状态）

#### 反向传播 — 直觉理解

```
前向传播：输入 → 经过网络 → 得到预测 → 算 Loss
反向传播：Loss → 反向计算每个参数对 Loss 的影响(梯度) → 更新参数

类比：
你在调游戏数值平衡，每个参数改动对"玩家体验评分"有多大影响。
反向传播就是自动告诉你：改哪个参数、改多少，能让评分提升最快。
```

#### 核心概念清单

| 概念 | 一句话解释 | 类比 |
|------|-----------|------|
| Loss Function | 衡量预测有多差 | 误差度量 |
| Gradient | 参数该往哪个方向调 | 梯度=方向导数 |
| Learning Rate | 每次调多大步 | 步长/学习速率 |
| Batch Size | 一次看多少样本再更新 | 批处理大小 |
| Epoch | 全部数据看一遍 | 全量遍历 |
| Overfitting | 在训练数据上过度拟合 | 过度针对测试环境优化 |
| Dropout | 随机关掉部分神经元 | 故障注入/混沌工程 |
| LayerNorm | 规范化每层输出 | 数据标准化 |

### PyTorch 完整训练模板

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader

# 1. 定义模型
class SimpleModel(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.1),
            nn.Linear(hidden_dim, output_dim),
        )

    def forward(self, x):
        return self.layers(x)

# 2. 准备数据
train_loader = DataLoader(dataset, batch_size=32, shuffle=True)

# 3. 初始化
model = SimpleModel(784, 256, 10).cuda()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
loss_fn = nn.CrossEntropyLoss()

# 4. 训练循环
for epoch in range(10):
    model.train()
    total_loss = 0
    for batch_x, batch_y in train_loader:
        batch_x, batch_y = batch_x.cuda(), batch_y.cuda()

        # 前向
        pred = model(batch_x)
        loss = loss_fn(pred, batch_y)

        # 反向
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()

        total_loss += loss.item()

    print(f"Epoch {epoch}, Loss: {total_loss / len(train_loader):.4f}")
```

**这套模板背下来**。后续微调 LLM 本质上也是这个结构，只是模型更大、数据更复杂。
